# The Cube v1.2.3

Release date: 2026-07-20

## Changed

- Added the selected Blender version artwork as a subtle blurred application background.
- Tuned the background to full width with a `-170px` vertical offset and `7px` blur.
- Kept the Library title and active-version banner fixed while browsing.
- Added independent scrolling for installed versions and the release catalog.
- Disabled accidental interface text selection while preserving selection in editable fields.

## Compatibility

- Features, Blender libraries, add-ons, presets, preferences, licenses, and updater workflows are unchanged.
